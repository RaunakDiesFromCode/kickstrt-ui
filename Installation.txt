IMPORTANT
 - this componet library only works for react+vite+tailwind
 - package should install if its not in packages.json else skip
 - create folders if not present

procedures :-
    npx kickstrt init
        - installs clsx, tailwind-merge, lucide-react
        - copies src/index.css and src/lib/utils.ts from this directory (in github) and to the directory user is installing it on in the same order replacing if necessary

    npx kickstrt init theme
        - check if Button is installed. if not install it.
        - copies src/components/theme-provider.tsx from this directory (in github) and to the directory user is installing it on in the same order replacing if necessary
        - copies src/components/UI/ThemeToggle.tsx from this directory (in github) and to the directory user is installing it on in the same order replacing if necessary

    npx kickstrt add <components>
        - check src/lib/componentMap.ts for the component's requirments and path from where it should be taken from this repo and installed on the user's.
        - package should install if its not in packages.json else skip
